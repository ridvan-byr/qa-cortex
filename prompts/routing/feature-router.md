# Feature Router

## Purpose

Identify which business feature the test is validating.

---

## Supported Features

Authentication

Search

Registration

Checkout

Profile

Settings

Navigation

Forms

CRUD

File Upload

Notifications

API

Payments

---

## Detection Sources

- Test title
- Describe block
- File name
- Selectors
- URLs
- Assertions
- API endpoints

---

## Rules

Choose the dominant feature.

Support multiple features when necessary.

Avoid guessing.

---

## Output

Primary Feature

Secondary Features

Confidence

Evidence

# Coverage Reviewer

## Purpose

Evaluate whether automated tests provide sufficient functional coverage.

---

## Evaluate

- Happy Path
- Negative Tests
- Boundary Tests
- Edge Cases
- Error Handling
- Data Variations
- Unicode
- Accessibility

---

## Missing Coverage

Detect missing:

- Invalid input tests
- Empty values
- Maximum values
- Minimum values
- Special characters
- Locale variations

---

## Output

Coverage Score

Missing Coverage

Recommendations

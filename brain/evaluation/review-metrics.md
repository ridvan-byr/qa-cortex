# Review Metrics

QA Cortex evaluates tests using multiple metrics.

---

# Metrics

- Coverage
- Quality
- Risk
- Confidence
- Maintainability
- Readability
- Assertion Strength
- Test Isolation
- Data Diversity
- Security Awareness

---

# Reporting Rules

Every review should include:

- Estimated Coverage
- Estimated Quality
- Risk Level
- Confidence Level
- Missing Areas
- Improvement Suggestions

---

# Principles

Metrics should guide improvement, not simply produce scores.

Recommendations must always explain which metric is affected.
